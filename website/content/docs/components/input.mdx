Single-line text input (themed LineEdit).

<ExampleFrame src="/godot-addon-shadcn/example/index.html?c=input" />

Use the panel on the left of the demo to customize this component live, and the
toolbar to switch base color, accent and light/dark.

## Usage

```gdscript
var le := LineEdit.new()
le.placeholder_text = "Email"
```
