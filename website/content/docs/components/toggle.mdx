<ExampleFrame src="/godot-addon-shadcn/example/index.html?c=toggle" />

Use the panel on the left of the demo to customize this component live, and the
toolbar to switch base color, accent and light/dark.

## Usage

```gdscript
var t := ShadcnToggle.new()
t.variant = ShadcnToggle.Variant.DEFAULT
t.text = "Bold"
```
