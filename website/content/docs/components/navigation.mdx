## ShadcnAccordionItem

```gdscript
var item := ShadcnAccordionItem.new()
item.title = "Is it accessible?"
item.body = "Yes. It adheres to sensible defaults."
item.expanded = true
add_child(item)
```

## ShadcnBreadcrumb

```gdscript
var bc := ShadcnBreadcrumb.new()
bc.items = ["Home", "Components", "Breadcrumb"]
bc.crumb_pressed.connect(func(i): print("crumb ", i))
```

## ShadcnPagination

```gdscript
var pg := ShadcnPagination.new()
pg.page_count = 10
pg.current = 3
pg.page_changed.connect(func(p): print("page ", p))
```

## ShadcnSidebar

```gdscript
var sb := ShadcnSidebar.new()
sb.expanded_width = 240
add_child(sb)
sb.content.add_child(my_nav_list)
sb.toggle()   # animate collapse/expand
```

## ShadcnCommand

```gdscript
var cmd := ShadcnCommand.new()
cmd.add_item("new", "New File")
cmd.add_item("open", "Open…")
cmd.selected.connect(func(id): print(id))
add_child(cmd)   # drop it inside a ShadcnDialog for a ⌘K palette
```

## ShadcnCarousel

```gdscript
var car := ShadcnCarousel.new()
for i in 3:
	var slide := my_slide(i)
	car.add_slide(slide)
add_child(car)   # prev/next + dots included
```

## ShadcnCard

```gdscript
var card := ShadcnCard.new()
var box := VBoxContainer.new()
card.add_child(box)
var title := ShadcnCardTitle.new(); title.text = "Create project"
var desc := ShadcnCardDescription.new(); desc.text = "Deploy in one click."
box.add_child(title); box.add_child(desc)
```

## Tabs & Separator

Themed `TabContainer` / `TabBar` and `HSeparator` / `VSeparator` work as usual.
