Hover tooltip that opens above the trigger.

<ExampleFrame src="/godot-addon-shadcn/example/index.html?c=tooltip" />

Use the panel on the left of the demo to customize this component live, and the
toolbar to switch base color, accent and light/dark.

## Usage

```gdscript
var tip := ShadcnTooltip.new()
tip.text = "Add to library"
my_button.add_child(tip)
```
