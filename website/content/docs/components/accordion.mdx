Collapsible titled section.

<ExampleFrame src="/godot-addon-shadcn/example/index.html?c=accordion" />

Use the panel on the left of the demo to customize this component live, and the
toolbar to switch base color, accent and light/dark.

## Usage

```gdscript
var a := ShadcnAccordionItem.new()
a.title = "Is it accessible?"
a.body = "Yes."
a.expanded = true
```
