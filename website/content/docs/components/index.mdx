title: Overview
description: Every shadcn component and how it maps to shadcn-godot.
---

Components come in two forms:

- **Custom nodes** — reusable `class_name Shadcn*` nodes for things Godot lacks.
  Create them in code (`ShadcnButton.new()`) or from the editor's *Create Node*
  dialog.
- **Themed stock controls** — Godot's own controls, restyled by the theme. Use
  them as you normally would; they just look like shadcn.

All custom nodes register in the `shadcn_refresh` group and expose a `refresh()`
method, so [`ShadcnTokens.apply()`](/docs/color-schemes) restyles them live.
