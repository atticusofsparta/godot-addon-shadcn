## ShadcnAlert

```gdscript
var a := ShadcnAlert.new()
a.variant = ShadcnAlert.Variant.DESTRUCTIVE  # DEFAULT or DESTRUCTIVE
a.title = "Error"
a.description = "Your session has expired."
add_child(a)
```

## ShadcnToast (Sonner)

Enabling the plugin installs a `ShadcnToasts` autoload. Or fire one from
anywhere:

```gdscript
ShadcnToast.notify(self, "Event created", "Sunday at 9:00 AM.")
# via the autoload:
ShadcnToasts.push("Saved", "Your changes were saved.")
```

## ShadcnSpinner / ShadcnSkeleton

```gdscript
var spinner := ShadcnSpinner.new()
spinner.diameter = 28

var sk := ShadcnSkeleton.new()
sk.custom_minimum_size = Vector2(180, 24)   # pulsing placeholder
```

## Progress

Themed `ProgressBar`:

```gdscript
var pb := ProgressBar.new()
pb.value = 60
pb.show_percentage = false
```

## ShadcnEmpty

```gdscript
var e := ShadcnEmpty.new()
e.icon_text = "📭"
e.title = "No projects"
e.description = "Create your first project to get started."
add_child(e)
var b := ShadcnButton.new(); b.text = "New project"
e.actions.add_child(b)    # actions is built in _ready, so add after add_child(e)
```
