Bottom drawer.

<ExampleFrame src="/godot-addon-shadcn/example/index.html?c=drawer" />

Use the panel on the left of the demo to customize this component live, and the
toolbar to switch base color, accent and light/dark.

## Usage

```gdscript
var dr := ShadcnDrawer.new()
dr.title = "Move goal"
add_child(dr); dr.open()
```
