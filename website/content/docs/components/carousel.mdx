Pager with prev/next and dots.

<ExampleFrame src="/godot-addon-shadcn/example/index.html?c=carousel" />

Use the panel on the left of the demo to customize this component live, and the
toolbar to switch base color, accent and light/dark.

## Usage

```gdscript
var car := ShadcnCarousel.new()
car.add_slide(my_slide)
add_child(car)
```
