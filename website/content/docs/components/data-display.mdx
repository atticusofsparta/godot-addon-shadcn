## ShadcnAvatar

```gdscript
var av := ShadcnAvatar.new()
av.diameter = 40
av.image = preload("res://user.png")   # falls back to initials if null
av.fallback = "CN"
```

## Table

Themed `Tree` / `ItemList` for simple tables and lists.

## ShadcnDataTable

A filter box plus a sortable `Tree`. Click a column header to sort; type to
filter.

```gdscript
var dt := ShadcnDataTable.new()
dt.set_columns(["Invoice", "Status", "Amount"])
dt.add_row(["INV001", "Paid", "$250"])
dt.add_row(["INV002", "Pending", "$150"])
dt.row_activated.connect(func(row): print(row))
add_child(dt)
```
