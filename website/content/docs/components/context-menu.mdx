Right-click menu (themed PopupMenu).

<ExampleFrame src="/godot-addon-shadcn/example/index.html?c=context-menu" />

Use the panel on the left of the demo to customize this component live, and the
toolbar to switch base color, accent and light/dark.

## Usage

```gdscript
var pm := PopupMenu.new()
pm.add_item("Cut"); pm.add_item("Copy")
```
