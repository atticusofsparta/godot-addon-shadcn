Use the panel on the left of the demo to customize this component live, and the
toolbar to switch base color, accent and light/dark.

## Usage

```gdscript
var card := ShadcnCard.new()
var box := VBoxContainer.new(); card.add_child(box)
var t := ShadcnCardTitle.new(); t.text = "Create project"
box.add_child(t)
```
