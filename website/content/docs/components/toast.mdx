Use the panel on the left of the demo to customize this component live, and the
toolbar to switch base color, accent and light/dark.

## Usage

```gdscript
ShadcnToast.notify(self, "Event created", "Sunday at 9:00 AM.")
# or via the autoload:
ShadcnToasts.push("Saved", "Your changes were saved.")
```
