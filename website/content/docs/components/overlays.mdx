The dialog/sheet/drawer nodes are `CanvasLayer`s — add one to the tree, then
`open()` / `close()`. They animate in over a dimmed backdrop.

## ShadcnDialog

```gdscript
var d := ShadcnDialog.new()
d.title = "Edit profile"
d.description = "Make changes to your profile here."
add_child(d)                       # add first so body/footer exist
var save := ShadcnButton.new(); save.text = "Save"
save.pressed.connect(d.close)
d.footer.add_child(save)
d.closed.connect(d.queue_free)
d.open()
```

## ShadcnAlertDialog

```gdscript
var a := ShadcnAlertDialog.new()
a.title = "Are you absolutely sure?"
a.description = "This permanently deletes your account."
a.destructive = true
a.action_text = "Delete"
a.confirmed.connect(func(): print("deleted"))
add_child(a)
a.open()
```

## ShadcnSheet / ShadcnDrawer

```gdscript
var s := ShadcnSheet.new()
s.side = ShadcnSheet.SheetSide.RIGHT   # LEFT, RIGHT, TOP, BOTTOM
s.title = "Settings"
add_child(s); s.open()

var dr := ShadcnDrawer.new()           # bottom drawer
dr.title = "Move goal"
add_child(dr); dr.open()
```

## ShadcnTooltip / ShadcnHoverCard

Add as a child of any `Control`. They open **above** the trigger by default.

```gdscript
var tip := ShadcnTooltip.new()
tip.text = "Add to library"
my_button.add_child(tip)

var hc := ShadcnHoverCard.new()
my_link.add_child(hc)
# populate hc.body after it's built (on first hover) or call hc._build() first
```

## Popover / Dropdown / Context menu

Themed Godot controls:

```gdscript
# Popover — a styled PopupPanel
var pop := PopupPanel.new()
pop.wrap_controls = true
pop.add_child(my_content)
add_child(pop)
pop.popup(Rect2i(button.get_screen_position() + Vector2(0, button.size.y + 6), Vector2i.ZERO))

# Dropdown / context menu — MenuButton / PopupMenu are themed automatically
var mb := MenuButton.new()
mb.get_popup().add_item("Profile")
```
