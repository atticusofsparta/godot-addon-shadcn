Animated pill toggle.

<ExampleFrame src="/godot-addon-shadcn/example/index.html?c=switch" />

Use the panel on the left of the demo to customize this component live, and the
toolbar to switch base color, accent and light/dark.

## Usage

```gdscript
var s := ShadcnSwitch.new()
s.switched.connect(func(on): print(on))
```
