---
title: Charts
description: One flexible ShadcnChart covering shadcn's chart gallery.
---

`ShadcnChart` is a single custom-drawn node that covers shadcn's chart gallery
through **options** rather than dozens of separate types. It animates in, shows
a hover data tooltip, and colors series from the palette's `chart-1…5`.

## Kinds

```gdscript
ShadcnChart.Kind.LINE
ShadcnChart.Kind.AREA
ShadcnChart.Kind.BAR
ShadcnChart.Kind.PIE
ShadcnChart.Kind.DONUT
ShadcnChart.Kind.RADAR
ShadcnChart.Kind.RADIAL
```

## Options

| Property | Applies to | Notes |
| --- | --- | --- |
| `curve` | line, area | `LINEAR`, `STEP`, `SMOOTH` |
| `horizontal` | bar | horizontal bars |
| `stacked` | bar, area | stack series |
| `gradient` | area | fade the fill |
| `show_dots` | line, area | point markers |
| `show_values` | bar, line | value labels |
| `x_labels` | all | category / axis labels |
| `show_grid`, `show_legend`, `y_ticks` | cartesian | |

## Example

```gdscript
var chart := ShadcnChart.new()
chart.kind = ShadcnChart.Kind.AREA
chart.curve = ShadcnChart.CurveType.SMOOTH
chart.gradient = true
chart.x_labels = ["Jan", "Feb", "Mar", "Apr", "May", "Jun"]
chart.add_series([186, 305, 237, 173, 209, 264], "Desktop")
chart.add_series([80, 200, 120, 190, 130, 140], "Mobile")
add_child(chart)
```

Recipes for the gallery:

```gdscript
# bar horizontal + stacked
chart.kind = ShadcnChart.Kind.BAR; chart.horizontal = true; chart.stacked = true
# line step (no dots)
chart.kind = ShadcnChart.Kind.LINE; chart.curve = ShadcnChart.CurveType.STEP; chart.show_dots = false
# radar / radial
chart.kind = ShadcnChart.Kind.RADAR
chart.kind = ShadcnChart.Kind.RADIAL
```

Charts follow the active accent automatically — switching scheme with
`ShadcnTokens.apply()` recolors them.
