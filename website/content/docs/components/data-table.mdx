Filterable, sortable table.

<ExampleFrame src="/godot-addon-shadcn/example/index.html?c=data-table" />

Use the panel on the left of the demo to customize this component live, and the
toolbar to switch base color, accent and light/dark.

## Usage

```gdscript
var dt := ShadcnDataTable.new()
dt.set_columns(["Invoice", "Status", "Amount"])
dt.add_row(["INV001", "Paid", "$250"])
```
