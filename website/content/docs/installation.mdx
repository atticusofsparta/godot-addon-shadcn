## Requirements

Godot **4.3+** (developed and verified against 4.6.3).

## Install with one command

Run this from your project's `addons/` folder:

```bash
cd addons
curl -fsSL https://raw.githubusercontent.com/atticusofsparta/godot-addon-shadcn/main/install.sh | bash
```

It downloads the latest `addons/shadcn` into the current folder. Pin a tag or
branch with `... | bash -s -- v0.1.0`. Then enable **shadcn-godot** in
*Project → Project Settings → Plugins*.

## Install manually

1. Copy the `addons/shadcn/` folder from this repository into your project's
   `addons/` directory.
2. Enable **shadcn-godot** in *Project → Project Settings → Plugins*.

```
your-project/
├── addons/
│   └── shadcn/          ← copy this folder
│       ├── plugin.cfg
│       ├── theme_builder.gd
│       ├── themes/
│       └── components/
└── project.godot
```

Enabling the plugin registers a convenience autoload, `ShadcnToasts`. Every
component also declares a `class_name`, so the nodes appear in the editor's
*Create New Node* dialog and are usable from code even with the plugin disabled.

## Apply the theme

Set one of the bundled themes on your root `Control` to theme the whole UI:

- `res://addons/shadcn/themes/shadcn_neutral_dark.tres`
- `res://addons/shadcn/themes/shadcn_zinc_light.tres`
- …one per base color × `light`/`dark`
- `shadcn_dark.tres` / `shadcn_light.tres` — neutral, default names

```gdscript
extends Control

func _ready() -> void:
	theme = load("res://addons/shadcn/themes/shadcn_dark.tres")
```

Prefer code? Build a theme for any base + accent + mode at runtime:

```gdscript
$Root.theme = ShadcnTheme.build("zinc", "blue", true)  # base, accent, dark
```

## Use a component

```gdscript
var btn := ShadcnButton.new()
btn.variant = ShadcnButton.Variant.DESTRUCTIVE
btn.text = "Delete"
add_child(btn)
```

See [Components](/docs/components) for the full catalog.
